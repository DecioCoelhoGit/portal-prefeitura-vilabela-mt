# portal-prefeitura-vilabela-mt
Plataforma digital institucional da Prefeitura de Vila Bela da Santíssima Trindade-MT, com foco em transparência, gestão inteligente e acesso à informação. Integra dashboard, notícias, cultura, turismo e dados públicos em arquitetura leve (HTML, CSS, JS), com visão moderna de governança digital.
